# Instruções de uso

1. Compile o programa Java:

   ```bash
   cd "c:\Users\carva\OneDrive\Área de Trabalho\oficina-3-contagem-de-opera-es-lucas-8844"
   javac src/App.java
   ```

2. Execute:

   ```bash
   java -cp src App
   ```

3. Verifique arquivos gerados:

   - `resultado.csv`: tabela de `algoritmo,teste,n,operacoes,ms`
   - `graficos_operacoes.png` e `graficos_tempo.png`: gráficos automáticos de `operacoes` e `tempo`

4. Crie gráficos adicionais em planilha usando `resultado.csv` se desejar.

5. Interprete comparando complexidades:

   - `codigo1`: O(n)
   - `codigo2`: O(n)
   - `codigo3`: O(n²)
   - `codigo4`: O(2^n)
