# Código do Projeto

Neste diretório está o código fonte da atividade de contagem de operações e análise de desempenho.

Arquivo principal:
- `App.java`: contém os 4 códigos de teste (`codigo1`, `codigo2`, `codigo3`, `codigo4`), o gerador de vetores e as funções de medição.

Saídas geradas pela execução:
- `resultado.csv` (salva em runtime)
- `graficos_operacoes.png` e `graficos_tempo.png` (salva em runtime)

Para rodar: `javac App.java && java App` a partir de `src` ou via `java -cp src App` a partir do diretório raiz do projeto.
